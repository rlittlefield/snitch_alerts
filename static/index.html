Carbon Snitch Alert System

Get alerts sent to your phone or browser
when someone hits your snitches

The CSAS is written by Ryan Littlefield. It is designed to alert you and your friends when someone hits a snitch in civcraft. Push alerts are currently handled through Pushbullet public channels.

To get started:

  1. Set up a TSV document on the web somewhere. I recommend using a google doc, and making a public URL for viewing. You can then download an export as a TSV and use that URL in the config.
  2. Get a pushbullet account
  3. Set up a pushbullet channel and copy the tag/name
  4. Find your pushbullet API token
  5. Determine what file path on your computer you would like to save all snitch hits to. This should be a full path, but will otherwise go into the same directory as monitor_snitches.py
  6. Find your .minecraft directory, and locate the log file. It should be something like C:\Users\Ryan\AppData\Roaming\.minecraft\logs\latest.log
  7. Fill out the information in the Setup section below
  8. Press "Start"

Config

We just need to fill out a few quick items before we can get started.

  1. What is the full path to your minecraft chat log?
  2. Full path to where you want to save the snitch logs (csv)
  3. PushBullet API token
  4. PushBullet Channel
  5. TSV export of badguy list url (google doc export url)

    You need to get the export url of the public facing google doc. This can be done in several ways, but the easiest is to go to the "share" link, get a public URL, then open an icognito window and go to that URL. Then export the thing as a TSV, and go to your chrome downloads page, right click on it, and copy the URL.
